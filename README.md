# 🏎️ F1 Insight

An interactive Formula 1 analytics dashboard built with **Python**, **Streamlit**, and **FastF1** that transforms race data into meaningful visual insights.

## ✨ Features

- 🏁 Race Leaderboard
- 🏆 Podium Results
- 📈 Lap Time Analysis
- 👥 Driver Comparison
- 🛞 Tyre Strategy Visualization
- 📊 Race Statistics
- 💡 Race Insights
- 🗺️ Circuit Maps

## 🛠️ Tech Stack

- Python
- Streamlit
- FastF1
- Plotly
- Pandas
- Git & GitHub

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/F1-Insight.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
